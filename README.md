# 🎓 Student Management System

A modern **full-stack Student Management System** built with the **MERN Stack** that simplifies student record management through an intuitive and responsive interface. The application enables administrators to efficiently manage student information, course enrollments, and enrollment dates while providing real-time updates for a seamless user experience.

---

## ✨ Features

- 👨‍🎓 Add, edit, delete, and view student records
- 📚 Manage course enrollments
- 📅 Track enrollment dates
- ⚡ Real-time UI updates
- 🔍 Search and organize student information
- 🌐 RESTful API integration
- 📱 Responsive and user-friendly interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Run the Application

Backend

```bash
npm start
```

Frontend

```bash
npm start
```
